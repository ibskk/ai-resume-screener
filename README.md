# AI Resume Screener

An AI-powered tool that analyzes how well a resume matches a job description using NLP similarity scoring.

## Features
- Upload resume as PDF
- Paste job description
- TF-IDF + cosine similarity match score
- Keyword gap detection for missing skills
- Streamlit web interface

## Tech Stack
- Python
- scikit-learn
- Streamlit
- pdfplumber
